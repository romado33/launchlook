Questions

FAQ

Which platforms do you support?+

Any vibe-coded web app with a public URL. We often review Lovable, Bolt, Base44, and Replit apps — also v0, Cursor-built apps, and others. If your builder outputs a shareable link, we can usually review it.

Is this only for mobile apps?+

No — we review web apps built with any vibe-coding tool (Lovable, Bolt, v0, Cursor, etc.). “Phone check” means we load your URL at mobile width in the browser, not the App Store.

What do you need from me?+

Your public or staging URL, a one-line description, and your builder. For Full Package ($29), two temporary test accounts so we can check data isolation.

How is this different from a security scanner?+

We focus on what visitors see — broken UI, placeholders, trust pages. For deep security, use something like VAS. Use both if you can.

Lovable already has SEO review — why this?+

SEO tools help Google. We help the friend you send the link to — dead buttons, dev tools on prod, missing legal pages, fake-looking reviews.

Staging URLs OK?+

Yes if they're reachable on the internet (Lovable preview, Vercel staging, etc.). localhost won't work.

How do I know my data is safe?+

We only access your app through the public URL and any temporary test accounts you provide. We don't request code access, database access, or API keys. Credentials are submitted through the intake form only and deleted within 24 hours of delivery. See our privacy policy.
