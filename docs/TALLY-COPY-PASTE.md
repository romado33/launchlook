# Tally intake — which file to use

Tally’s editor does **not** accept markdown. Use the **plain text** files below.

| File | Use when |
|------|----------|
| **[`TALLY-AI-ONE-SHOT.txt`](TALLY-AI-ONE-SHOT.txt)** | **Fastest** — paste into Tally **Create form with AI**, then fix conditionals |
| **[`TALLY-INTAKE-PASTE.txt`](TALLY-INTAKE-PASTE.txt)** | Build by hand — each question has `>>> PASTE START` / `<<< PASTE END` blocks |
| **[`TALLY-THANK-YOU-PASTE.txt`](TALLY-THANK-YOU-PASTE.txt)** | Thank-you message only (Settings → Thank you page) |
