# Cold outreach Loom script (Week 1)

## 60-second Loom structure

**0:00–0:10** — Show their app in browser. "Hey [name], I clicked through [app name] like a first-time visitor."

**0:10–0:35** — Show 2–3 specific issues (dead button, placeholder text, missing privacy, dev tools visible). No jargon.

**0:35–0:50** — "I run LaunchLook — a pre-launch checkup for vibe-coded apps. Plain-English report + copy-paste fix prompts for your builder."

**0:50–1:00** — CTA: "Starter Package is $9 if you want the full list. Link: launchlook.app"

## DM text (pair with Loom)

```
Hey [name] — I recorded a 60-sec walkthrough of [app] before you share it wider.
[LOOM LINK]

I flagged [issue 1] and [issue 2]. LaunchLook does a full pass for $19 (Starter Package) with fix prompts you paste into [Lovable/Bolt/etc].

Want 3 free findings first? Drop your URL at launchlook.app — we'll email the top 3 when they're ready (usually within a few days), no credit card.

No pressure — launchlook.app if useful.
```

## Rules

- **Never use the word "security" as the lead.** Polish + placeholders + sharing risks.
- **Never say "audit"** — say "checkup" or "second pair of eyes."
- Personalize the first line with something specific you saw.
